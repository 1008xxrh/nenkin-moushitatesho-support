# 病歴・就労状況等申立書 サポート（試作 第1〜5章 + 過去編）

障害年金の **病歴・就労状況等申立書** を書くための**現状整理＋過去編整理ツール**です。

公開ページ：  
[https://1008xxrh.github.io/nenkin-moushitatesho-support/](https://1008xxrh.github.io/nenkin-moushitatesho-support/)

---

## このツールの目的

申立書を書く前に「今どんな状況か」「過去に何があったか」を、**認知負荷をできるだけ下げながら整理**するための試作版です。

特に以下の負担を軽減することを目指しています：
- 何を書けばいいかわからない
- 時系列が頭の中でぐちゃぐちゃになる
- 現在の困りごとと過去の出来事が混ざって整理しにくい

---

## 現在できること（主な機能）

### 1. 現状整理（第1〜2章）
- 現在の就労状況（働いている／働いていない）
- 仕事内容・通勤・出勤日数・勤務中の体調
- 日常生活の困りごと（ADL + その他）
- 不就労理由の整理

### 2. 年表・出来事の整理（第3〜4章）
- 生活や就労に影響した出来事をカード形式で入力
- 年齢帯に配置して年表化
- 通院期間・通院していない期間の整理
- 病院情報の登録

### 3. 認定日頃の状況（任意・第5章）
- 障害認定日頃の就労状況・日常生活状況を別途入力可能
- 入力しなくても空欄で出力可能

### 4. 障害者手帳情報（任意）
- 手帳の種類・等級・交付年月日・障害名

### 5. 過去編（断片＋病院）
- 「いつ頃かわからない」思い出も気軽に置ける断片機能
- 過去の病院登録（時期・確かさ・メモ）
- 時期ごとの見返し機能

### 6. プレビュー＆Excel出力
- 紙面を模したリアルタイムプレビュー
- 公式様式（02.xlsx）への一部自動書き出し
- 申立書の下書き確認画面

---

## まだ対応していないこと（今後の拡張予定）

- 傷病名・発病日・初診日の詳細整理
- 完全な続紙（04など）対応
- PDF出力
- 診断書との完全連動機能

---

## 保存について

入力内容は**ブラウザの localStorage** に保存されます。
- 外部サーバーには一切送信されません
- 同じ端末・ブラウザで継続して使えます
- 他の端末やシークレットモードでは消えます

---

## 想定している使い方

1. 現在の生活・就労状況を整理（第1〜2章）
2. 過去の出来事・病院を少しずつ置いていく（第3〜4章＋過去編）
3. 認定日頃の状況を手書き感覚で入力（任意）
4. プレビューで確認 → Excelダウンロード
5. 必要に応じて年金事務所・社労士さんに相談しながら完成
