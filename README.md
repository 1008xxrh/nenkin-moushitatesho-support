# 病歴・就労状況等申立書 サポート（現状整理版MVP）

障害年金の **病歴・就労状況等申立書** を書く前段階として、
**現在の生活状況・就労状況・通院状況・これまでの出来事** を、質問に答えながら整理するための試作版です。

公開ページ：
https://1008xxrh.github.io/nenkin-moushitatesho-support/

---

# この試作版の目的

病歴・就労状況等申立書は、

* 何を書けばよいのかわかりにくい
* 時系列の整理が難しい
* 文章化の負担が大きい
* 「今の困りごと」と「過去の通院歴・出来事」が頭の中で混ざりやすい

といった負担が大きい書類です。

この試作版では、いきなり申立書全体を完成させるのではなく、まず

* 今どんな生活上の困りごとがあるか
* 今どんな働き方・働きにくさがあるか
* どんな通院歴があるか
* 生活や就労に影響した出来事をどう並べるか

を、**認知負荷をなるべく下げながら整理すること**を目的にしています。

---

# この試作版でできること

現在の試作版では、主に次の整理ができます。

## 1. 就労状況の整理

* 現在働いているかどうか
* 働いている場合の勤務状況
* 働いていない場合の理由
* 通勤状況や勤務上の配慮など

## 2. 現在の生活状況の整理

* 日常生活の困りごと
* ADL（日常生活動作）に関する状況
* 「その他、日常生活で不便なこと」の整理

## 3. 出来事の整理

* 生活や就労に影響した出来事をカードとして入力
* 出来事を年表上に配置して整理

## 4. 通院状況の整理

* 通院した病院・期間の整理
* 通院していない期間（通院空白期間）の整理
* 通院空白の理由の整理

## 5. Excel出力

入力した内容の一部を、**病歴・就労状況等申立書の様式（02.xlsx）の一部**に書き出せます。

---

# この試作版でまだ対応していないこと

この試作版は **「現状整理版MVP」** です。
そのため、以下の項目にはまだ対応していません。

## 1. 傷病名・発病時期・初診日などの過去編

以下のような、**診断書や受診状況等証明書との整合が必要な情報**は、現在は未対応です。

* 傷病名
* 発病した時期
* 初診日
* 初診に至る経緯の詳細整理
* 発病初期〜初診頃の時系列整理

これらは、現在の生活状況や通院状況の整理とは性質が違い、
確認しながら慎重に埋める必要があるため、現状整理パートとは分けて扱う方針です。

## 2. 申立書の全項目の出力

現在のExcel出力は、**様式（02.xlsx）の一部のみ**に対応しています。
申立書の全項目が自動で埋まるわけではありません。

## 3. PDF出力

現在は PDF 出力には対応していません。

## 4. 続紙（04など）を含む完全出力

通院歴や出来事が多い場合の続紙対応など、完全な出力機能はまだ未実装です。

# 出力に関する注意

## 通院期間・通院空白期間の出力件数について

現在の試作版では、Excelに書き出される通院関係の期間は、

**病院の通院期間 + 通院していない期間 を合わせて最大5件まで**

です。

6件目以降がある場合は、この試作版では出力されません。
必要に応じて、Excel出力後にご自身で追記してください。

## 出力対象外の項目について

傷病名・発病時期・初診日など、診断書等との整合が必要な項目は、
この試作版の出力対象に含まれていません。

---

# 保存について

この試作版では、入力した内容は **ブラウザの localStorage を使って、お使いの端末内に保存** されます。

* 入力内容を外部サーバーへ送信することは想定していません
* サーバー側で個人情報を保存する仕組みは入れていません
* ただし、**同じ端末・同じブラウザ上にデータが残る**ため、共有PCなどでは注意してください

必要に応じて、画面上の「最初からやり直す」導線から保存内容を消去してください。

---

# 想定している使い方

この試作版は、**申立書をその場で完成させるツール**というより、

1. まず現在の生活・就労・通院状況を整理する
2. 出来事を時系列に並べる
3. 申立書に書く土台を作る
4. その後、必要に応じて診断書等と照らし合わせながら過去編を整理する

ための、**前半の整理ツール**として作っています。

# フィードバック歓迎です

この試作版は、実際に使ってみて

* どこで疲れたか
* どこで迷ったか
* どこで止まったか
* 何がわかりにくかったか
* どの質問が答えにくかったか
* 思い出せなくて詰まった箇所はどこか
* 「今のことを整理する流れ」として無理がなかったか

を知るために公開しています。

特に、以下のような観点の感想があると助かります。

* 面倒に感じた箇所
* 何を聞かれているのかわからなかった箇所
* 聞いてほしかったのに聞かれなかった項目
* 逆に、聞かれてしんどかった項目
* 全体でどれくらい時間がかかったか

良い点・悪い点どちらも歓迎です。

# 現在の位置づけ

このリポジトリで公開しているものは、**障害年金申立書ゲーム化プロジェクトの Phase5 実装中のMVP**です。

現時点では、

* まず「現状整理版」を成立させる
* 実際に使ったときの認知負荷や詰まりポイントを確認する
* その後に、過去編や出力範囲の拡張を検討する

という段階にあります。

この試作版は、障害年金の申立書作成を補助するための試作ツールです。
申立書の記載内容そのものの正確性や、制度上の判断を保証するものではありません。

実際の提出にあたっては、必要に応じて

* 年金事務所
* 社会保険労務士
